#Hospital Management System

The Hospital Management System (HMS) is a web-based application developed using JavaScript and MySQL to manage and automate daily hospital operations efficiently. The system helps in handling patient records, doctor information, appointments, and billing in a centralized and secure manner.

This project reduces manual paperwork, improves data accuracy, and provides quick access to hospital information. It is ideal for academic projects and for learning full-stack web development with database integration.


#Key Features

User authentication (Admin / Staff login)

Patient registration and medical records management

Doctor and staff management

Appointment scheduling system

Billing and payment management

Secure data storage using MySQL

Responsive and user-friendly interface

#Technologies Used

Backend: JavaScript 

Database: MySQL



#Objective

The main objective of this system is to digitize hospital processes, reduce manual effort, and improve operational efficiency while ensuring data security and reliability.
